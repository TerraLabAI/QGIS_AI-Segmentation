about=AI Segmentation brings state-of-the-art AI to QGIS. Powered by Meta's Segment Anything Model (SAM), it lets you segment any object in satellite imagery, orthophotos, or raster datasets with a simple click.

    FEATURES

    - One-Click Segmentation: Left-click to include, right-click to exclude
    - Auto Setup: Creates isolated Python environment with all dependencies
    - Hardware Optimized: Runs on CPU, NVIDIA GPU (CUDA), or Apple Silicon (MPS)
    - Vector Export: Save segmentations as polygon layers
    - Cached Encoding: Fast subsequent sessions on same raster

    HOW TO USE

    1. Install Dependencies (first time only, ~2.5 GB)
    2. Download SAM Model (~375 MB)
    3. Select a raster layer
    4. Click "Start AI Segmentation"
    5. Click on objects to segment them
    6. Press S to save, then Export to Layer
